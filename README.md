# microdesafio_ml
Proyecto Mercado Liebre: Se diseña una Página web similar a un e-commers. Se crea un home, una página de register(crea tu cuenta)  y otra de login.  Se aplican conceptos de flexbox, formularios, , diseño responsive con medias queries, animation, position, funcionamiento y puesta en marcha con express, nodemon, librerias de fuentes y colores para el estilo de la pagina. 
Se utilizan buenas prácticas de programación, versionado con Git y deploy con HEROKU.
